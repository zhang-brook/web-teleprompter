# 网页提词器 · Web Teleprompter

> [English](README.md) | 中文

**一台会自己滚的提词器 —— 让你盯着镜头照稿念，双手彻底解放。**

💡 **一句话说明它是做什么的：** 一个免费、纯网页的提词器，能按固定速度自动滚动，或**跟着你的声音实时滚动**——适合直播带货、录课教学、短视频口播、演讲汇报等一切「照稿说」的场景。

🆓 **免费 · 零安装 · 免注册** —— 打开浏览器就能用。

> 🔗 **[在线演示](https://zhang-brook.github.io/web-teleprompter/)** &nbsp;·&nbsp; 💾 **[项目仓库](https://github.com/zhang-brook/web-teleprompter)**

## ✨ 为什么你会喜欢它（亮点功能）

- **🎙️ 语音跟随滚动**：开口说话，文稿自动滚到你正在读/说的位置并居中。自动处理停顿、换气、甚至换种说法，并且**在静音/换气后会自动重启识别继续跟随**，双手彻底解放，不用碰鼠标键盘。
- **🛡️ 开始前的智能语言检测**：进入语音模式前，自动扫描文稿——若发现是多种语言混合，或与你所选识别语言不一致，会弹窗提醒，并展示各语言的占比，让你一眼确认选对语言，避免口播中途读错。
- **🔄 任意角度旋转 + 翻转**：支持水平/垂直翻转，以及 0°/90°/180°/270° 一键切换或任意角度微调。无论手机横放、提词器镜像摆放还是侧着读，画面都能转过去对准你。
- **📍 朗读线 + 永不读串行**：一条高亮「当前读到哪」的参考线，可随手拖到舒服的高度（口播中也能拖）；朗读时当前字会高亮、已读过的文字自动变淡，眼睛永远对准正确的位置。
- **🖱️ 不晕的滚动**：鼠标滚轮滚动带缓动并限制单次幅度，不会一滚就飞很远找不回来；没开始时也能用鼠标/触摸上下翻看文稿。
- **🪟 怎么用都行 · 多窗口模式**：浮窗（可拖到任意位置、拉角缩放）、窗口全屏（填满屏幕）、屏幕全屏（进入浏览器全屏，干干净净只剩稿子），以及全屏浮窗（浏览器全屏的黑底上以浮窗形式展示提词器，适合在干净背景上演示）。
- **📐 排版细节随便调**：字号、行距（1.0–3.0）、字间距（0–20px）、字体、文字/背景颜色全部可调；还能一键清理连续空行，让稿面清爽好读。
- **💾 配置自动保存 + 导入导出**：你的所有设置自动存在本地，刷新不丢；也能一键导出/导入 JSON，换电脑、换浏览器无缝衔接。
- **📄 拖入 txt 即导入**：把 `.txt` 文稿文件直接拖进页面，立刻变成提词内容，无需复制粘贴。
- **🌐 面向全球**：界面支持简体中文 / 繁體中文 / English，语音识别支持 10 种语言（见下文）。

## 🚀 三步上手

1. **打开[在线演示](https://zhang-brook.github.io/web-teleprompter/)**（或本地运行，见下方）。无需账号、无需安装。
2. **粘贴或拖入文稿**：在左侧面板粘贴文字，或直接把 `.txt` 文件拖进页面。
3. **点「开始」**：选择「固定速度」（设定舒服的滚动节奏）或「语音跟随」（直接开念），自然照稿读即可。

就这么简单，其余都是可选的微调。

> 想在自己电脑上跑？需 **Node.js 22+**：
> ```sh
> pnpm install
> pnpm dev        # 本地开发预览
> pnpm build      # 类型检查 + 构建到 dist/
> ```
> 想要一份自己的在线版本？仓库已内置 GitHub Actions 工作流——推送到 `main` 即自动构建并发布到 GitHub Pages。
> ⚠️ **首次部署前必看**：请先进入仓库 **Settings → Pages → Build and deployment → Source**，手动把部署源切到 **GitHub Actions**，否则 Actions 首次发布时会报错 `Create Pages site failed. Error: Resource not accessible by integration`。

## 📋 功能一览

### 滚动方式
- **固定速度滚动**：自定义滚动速度（10–400 像素/秒），可随时暂停 / 继续。
- **语音跟随滚动**：基于浏览器内置语音识别，你念到哪、稿子滚到哪；识别到的是「增量」还是「重说」，都会智能对齐，不会跳回开头；在换气、停顿后会静默重启识别，持续跟随。
- **开始 / 停止**，运行中可**暂停 / 继续**。
- **识别实时回显**：语音跟随时，底部会实时显示当前「听到」的文字，一眼确认麦克风是否正常工作。

### 语音识别
- **选择识别语言**：内置 10 种——简体中文（普通话）、繁體中文（國語）、English (US)、English (UK)、日本語、한국어、Français、Deutsch、Español、Русский。
- **智能开机检测**：若文稿为多种语言混合，或语言与所选识别语言不一致，会弹出带各语言占比的提示框。可点「返回选择语言」修正，或「继续开始」执意进行。
- **加载动画**：语音引擎准备期间显示加载动画，让你知道正在就绪。

### 阅读辅助
- **朗读线**：当前阅读位置会对齐到一条高亮参考线，口播时也能直接拖动悬浮窗里的朗读线上下调整；还可在设置面板里精确设定它的高度（距顶部 5%–95%）。
- **当前位置高亮 + 已读变淡**：两种模式下，正在读的字会高亮、已读过的文字自动变淡，眼睛始终锁定正确位置。
- **滚轮缓动**：滚动带平滑缓动并限制单次幅度，手感顺、不晕；**未开始滚动时也能用鼠标/触摸上下浏览文稿**。
- **触摸 / 鼠标拖拽浏览**：移动端和桌面端都能直接拖着看前后文。
- **滚轮灵敏度可调**：自定义每滚一格文字移动多远（10–300 像素），贴合你的习惯，而不是一动就飞出去。

### 画面与排版
- **字体**：内置 5 款常用字体（系统默认 / 黑体 / 宋体 / 楷体 / 等宽），一键切换。
- **字号、文字颜色、背景颜色**：随心调成你最舒服的大小与对比度。
- **行距 / 字间距**：行距（1.0–3.0）与字间距（0–20px）可调，远距离、手机小屏都看得舒服。
- **水平翻转 / 垂直翻转**。
- **旋转**：0°/90°/180°/270° 快速切换，或任意角度（0–359°）微调，适配各种拍摄与摆放角度。
- **英文单词是否断开**：默认整词换行；勾选后，行尾放不下的英文单词可拆开放两行，避免长单词溢出。
- **移除连续空白行**：默认开启，自动把连续空行压缩为最多一个，稿面干净不空荡。

### 窗口与显示
- **浮窗模式**：标题栏（或可见的半透明拖拽手柄）可拖拽移动，右下角可拖拽缩放，自由摆放在屏幕任意位置。
- **窗口全屏**：填满主区域，不被设置面板挤占。
- **屏幕全屏**：调用浏览器全屏，隐藏一切干扰，只留稿子。
- **全屏浮窗**：进入浏览器全屏，但提词器以浮窗形式展示在干净的黑底上，适合在舞台上做无干扰演示。
- **主题**：深色 / 浅色 / 跟随系统，自动适配系统配色。
- **开始后自动隐藏设置面板**，进入纯净口播视图；桌面端设置面板可随时收起/展开。
- **响应式适配**：自动适配手机 / 平板 / 电脑；屏幕过窄时设置面板变为侧滑抽屉，提词器不会被挤到屏幕外。

### 文稿与配置
- **直接输入文稿**：在设置面板文本框中编写。
- **拖入 `.txt` 文档导入**：把文本文件拖进页面即可。
- **配置本地自动保存**：所有设置写入浏览器本地，刷新或重开不丢失。
- **导出 / 导入配置（JSON）**：备份设置，或在不同设备间迁移。
- **一键恢复默认设置**。

### 语言与界面
- **界面语言**：简体中文 / 繁體中文 / English，可在顶部导航栏随时切换。
- **语音识别语言**：共 10 种（见上文）。注意：浏览器一次只能识别单一语言，所以一次口播请保持文稿为单一语言。

## ❓ 注意事项

- **语音跟随**依赖浏览器的语音识别能力，建议使用 **Chrome / Edge** 并授予**麦克风权限**；部分浏览器或隐私模式下可能不可用（此时会提示切换为固定速度模式）。在桌面端 Chrome/Edge 上运行流畅，部分手机浏览器的语音引擎表现可能不一致。
- 设置保存在**当前浏览器**本地；换浏览器或清除站点数据会丢失，重要配置请用「导出配置」备份。
- 在线演示地址需由仓库启用 GitHub Pages 后生效（默认推送到 `main` 即自动发布）。
