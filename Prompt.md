# Prompt

帮我在这个项目下创建一个网页版的提词器，支持输入文本后点击开始按照固定速度/滚动跟随语音识别（重点），支持口播是外接鼠标，在口播滚动的情况下，可以用鼠标上下额外滚动。需要适配手机/平板/电脑尺寸，口播界面支持调整窗口位置、大小（相当于是网页中的一个漂浮窗，能够拖拽调节位置，拖右下角调大小），文字颜色、文字大小、字体，水平翻转/镜像翻转/水平加镜像翻转/90度、180度、270度旋转（及任意角度旋转）

在开始之前，如果你有任何疑问，请先向我提出确认，都了解清楚细节后再开始写代码

---

语音识别采用 浏览器内置 Web Speech API
滚动跟随语音识别实现方式 逐字匹配+自动居中，但有时会有气口，或者个别句子换种表述，需要能够机灵匹配（也就是说需要容忍一部分不匹配情况）
技术栈 Vue + Vite，我已经在当前项目下创建好了Vue空白项目
